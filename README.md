# three-fbx-viewer
Drag-and-drop preview forFBX models in WebGL using three.js. 
Demo https://gltf-viewer.donmccurdy.com/

## Quickstart

### Web

```
npm install
npm run dev
```

### Desktop (Electron)

To build the desktop application, run:

```shell
# development build
npm run dev:electron

# package for release
npm run package
```
