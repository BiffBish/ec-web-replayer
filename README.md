# three-fbx-viewer
Drag-and-drop preview for FBX models in WebGL using three.js.

Demo https://limitless.pro/three-fbx-viewer

## Quickstart

### Web

```
npm install
npm run dev
```

### Desktop (Electron)

To build the desktop application, run:

```shell
# development build
npm run dev:electron

# package for release
npm run package
```
